# Keiko Corp - Web Optimization Challenge

A web optimization project demonstrating performance best practices for modern websites.

## Overview

Landing page for Keiko Corp with focus on performance optimization techniques including:

- Image optimization (WebP format, responsive images)
- Font loading optimization with `font-display: swap`
- CSS/JS minification and bundling
- Critical CSS inlining
- Resource preloading
- Responsive design
- Email subscription integration

## Tech Stack

- Bootstrap
- Owl Carousel
- jQuery
- Mailchimp integration
